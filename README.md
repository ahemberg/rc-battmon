## Battmon
Arduino code for a transmitter and a receiver. The transmitter is connected to the balancing pins of a lipo and monitors the voltage. Then transmits this over RF

The receiver displays this data on an LCD-display

### BOM

Transmitter
  1. Arduino Pro Trinket
  2. Velleman VMA322
  3. 33 uF Electrolytic cap
  4. Resistors (as accurate as possible): 1x 15 kOhm, 1x 68 kOhm, 4.7 kOhm, 10k Ohm, 18 kOhm, 8.2 kOhm)
  4. Wires and connectors as applicable
  
 Receiver
  1. Arduino Uno or Duemlianove
  2. Velleman VMA322
  3. 4x20 LCD Display
  4. Optional Potentiomenter for contrast
  5. Optional Resistor for backlight
